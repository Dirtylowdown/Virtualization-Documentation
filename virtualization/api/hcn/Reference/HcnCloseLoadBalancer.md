# HcnCloseLoadBalancer

## Description

Close a handle to a load balancer.

## Syntax

```cpp
HRESULT
WINAPI
HcnCloseLoadBalancer(
    _In_ HCN_LOADBALANCER LoadBalancer
    );
```

## Parameters

`LoadBalancer`

Handle to a LoadBalancer [`HCN_LOADBALANCER`](./HCN_LOADBALANCER.md)

## Return Values

The function returns [HRESULT](./HCNHResult.md).
