# HCN\_CALLBACK

## Description

Handle to a callback registered on a hns object.


## Syntax

```cpp
typedef void* HCN_CALLBACK;
```


## Requirements

|Parameter|Description|
|---|---|

| **Minimum supported server** | Windows Server 2019 |

| **Header** | ComputeNetwork.h |
