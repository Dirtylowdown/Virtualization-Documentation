# HcnCloseNetwork

## Description

Close a handle to a Network.

## Syntax

```cpp
HRESULT
WINAPI
HcnCloseNetwork(
    _In_ HCN_NETWORK Network
    );
```

## Parameters

`operation`

Handle to a network [`HCN_NETWORK`](./HCN_NETWORK.md)

## Return Values

The function returns [HRESULT](./HCNHResult.md).
