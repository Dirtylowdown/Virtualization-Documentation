# HCN\_PORT\_RANGE\_RESERVATION

## Description

A range of ports for a reservation.

## Syntax

```cpp
typedef struct tagHCN_PORT_RANGE_RESERVATION
{
    // start and end are inclusive
    USHORT startingPort;
    USHORT endingPort;
} HCN_PORT_RANGE_RESERVATION;

```
