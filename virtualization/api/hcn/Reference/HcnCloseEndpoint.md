# HcnCloseEndpoint

## Description

Close a handle to an Endpoint.

## Syntax

```cpp
HRESULT
WINAPI
HcnCloseEndpoint(
    _In_ HCN_ENDPOINT Endpoint
    );
```

## Parameters

`Endpoint`

Handle to an endpoint [`HCN_ENDPOINT`](./HCN_ENDPOINT.md)

## Return Values

The function returns [HRESULT](./HCNHResult.md).
